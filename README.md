# MediCare Pro - Final Linked Clinic Flow

Run on Mac:
1. `cd` into this folder
2. `npm install`
3. `clang clinic_store.c -o clinic_store`
4. `chmod +x clinic_store`
5. `kill -9 $(lsof -ti:3000)` if port 3000 is busy
6. `npm start`
7. Open `http://localhost:3000/MediCare_Pro.html`

Password: demo123

Flow fixes:
- Doctor opens an upcoming appointment and consultation auto-fills patient, doctor, visit and appointment info.
- Doctor only enters consultation findings/diagnosis/medication.
- Saving consultation marks that appointment Completed.
- Invoice is generated by selecting a completed appointment, not by manually typing patient name.
- Invoice date defaults to the appointment date, so old appointment invoices do not count as today's revenue.
- Receptionist appointment status supports Scheduled, In Consultation, Completed, Cancelled and Missed.
